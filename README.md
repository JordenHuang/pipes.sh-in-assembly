# Pipe.asm Usage

## Build

Use `make` command to execute the Makefile

```bash
make
```

## Run

### Fullscreen

```bash
./pipe.out $(tput cols) $(tput lines)
```

### Custom width and height

```bash
./pipe.out [custom width] [custom height]
```

## Quit program

Just hit Ctrl+c
